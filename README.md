# Контрольная работа №3

## 📋 Выполненные задания

### Задание 6.1
Реализована защищенная базовой аутентификацией конечная точка FastAPI /login, которая принимает GET-запросы и возвращает секретное сообщение при правильных учетных данных.

### Задание 6.2
Реализована безопасная аутентификация с хешированием паролей (bcrypt) и защитой от тайминг-атак. Добавлены эндпоинты POST /register и GET /login.

### Задание 6.3
Добавлено управление документацией в зависимости от режима окружения (DEV/PROD). В DEV режиме /docs и /openapi.json защищены базовой аутентификацией, /redoc скрыт. В PROD режиме вся документация отключена (404).

### Задание 6.4
Реализована JWT аутентификация: POST /login выдает JWT токен, GET /protected_resource требует Bearer токен в заголовке Authorization.

### Задание 6.5
Расширена JWT аутентификация: добавлен POST /register, хеширование паролей, rate limiting (1 запрос/мин для /register, 5 запросов/мин для /login), защита от тайминг-атак.

### Задание 7.1
Реализован RBAC с ролями admin/user/guest. Администратор имеет полные права, пользователь может читать и обновлять, гость - только чтение.

### Задание 8.1
Создана таблица users в SQLite с полями id, username, password. Реализован эндпоинт POST /register для сохранения пользователей.

### Задание 8.2
Реализован полный CRUD для Todo-списка с использованием SQLite: создание (POST /todos), чтение (GET /todos/{id}), обновление (PUT /todos/{id}), удаление (DELETE /todos/{id}).

## 🚀 Установка и запуск

```bash
# 1. Клонирование репозитория
git clone <repository-url>
cd fastapi_auth_project

# 2. Создание виртуального окружения
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows

# 3. Установка зависимостей
pip install -r requirements.txt

# 4. Настройка окружения
cp .env.example .env

# 5. Запуск приложения
uvicorn main:app --reload
